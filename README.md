# yummy
Searchable repository of recipes we frequently use and are yummy.

## Development

### NodeJS dependencies

```
npm i
```

### Working with gatsby

Start development server
```
gatsby develop
```


Preview website in production mode

```
gatsby build
gatsby serve
```
Update website

```
npm run deploy
```
