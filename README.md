# yummy
Searchable repository of recipes we frequently use and are yummy.

## Development

### ImageMagick

On Ubuntu

```
$ apt-get install imagemagick
```

On Mac OS X

```
$ brew install imagemagick
```

On CentOS

```
$ yum install imagemagick
```

### NodeJS dependencies

```
npm i
```

### Working with hexo

Start hexo server
```
npm start
```

Build production version

```
npm run build
```
